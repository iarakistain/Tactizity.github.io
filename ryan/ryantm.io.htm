KMeans-JS

Machine Learning | Javascript | OpenSource
Visit Website

An implementation of the KMeans++ clustering algorithm in JavaScript. It can be used directly in the browser to learn natural groupings of data, and assign new data points to pre-determined categories. This implementation uses multithreading and the MapReduce paradigm to parallelize the computation. The image shown uses the library to cluster areas of the city into similar categories based on 22 demographic factors.

Demographics

Mapping | Census | Data Visualization
View Images

Knowing the demographic makeup of an area and considering the people who live there is just as important to design as knowing its physical characteristics. The US Census provides an enormous amount of demographic information down to a very fine level of geographic detail, however the interface for retrieving that information can often leave something to be desired. Using Python and SQL, millions of data points were collected at the block-group and tract level across the entire country, producing an interactive map that displays demographics of a site both absolute and relative to its county, and can visualize those in a number of ways.

Existing Building Data

Mapping | Vector Tiles | Buildings | Python
Visit Website

Using surveyors data of building footprints, combined with arial LiDAR scans,3d geometries were created that represent every building in the County. The data was then enhanced with information from the county assessor giving metadata to each building, such as use, size, number of residences, and more. Finally, the resulting database was cut into 3d vector tiles and served onto a web map by a custom tile server, allowing detailed information, visualization, and exportable context models for any site.

Development Pipeline

Mapping | Development | NodeJS
Visit Website

If you want to get the full picture of development in Portland, counting the cranes is insufficient. There are many projects in various stages of the cities pipeline, going through design review, getting permits, that are invisible. Using geospatial programming in NodeJS, weekly development reports from the city are overlaid on a grid where cells can accumulate and lose projects over time. By slicing this grid along its time dimension we created a constantly updating animation of where development is happening over time in Portland, and how that's changing.

Renters/Owners

Python | Big Data | Vector Tiles | Mapping
Visit Website

Understanding the patterns and structure in rental vs owned housing is integral to understanding the potential of a market, and watching these patterns change over time can give insight on where they will go in the future. Using Python, Futhark, and high performance GPU computing, census population data was transformed to generate a single map point for every single person in the United States, accurate down to the block they live on. As the points were generated, they were categorized as renters or owners based on census proportions. The resulting map required custom vector tiles to display the sheer density of information, and is a fascinating look at the density and organization of our cities.
